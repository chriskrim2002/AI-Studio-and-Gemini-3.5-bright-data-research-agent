# AI-Studio-and-Gemini-3.5-bright-data-research-agent
Autonomous market research agent powered by Gemini 3.5 Flash and google-genai SDK. It runs a ReAct loop using Bright Data's SERP &amp; Web Unlocker APIs to autonomously search, scrape, and synthesize live web data. Input any company to generate structured markdown reports detailing founders, funding, and news with zero manual steps.
